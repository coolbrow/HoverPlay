HoverPlay
=========

HoverZoom for audio files

##2 Modes

* [Hover Mode](#hover)
* [Replace Mode](#replace)


###Hover Mode
<a href='#hover'></a>
Just hover over a link to an audio file and it will start playing immediately!

###Replace Mode
<a href="#replace"></a>
Replaces all links to audio with the HTML5 audio player.
